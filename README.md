📌 Project Overview
Natours is a modern, responsive landing page for a fictional tour company. This project focuses on advanced CSS techniques, Sass architecture, animations, and responsive design without using any CSS frameworks.
Built as part of an Advanced CSS course by Jonas Schmedtmann.

🚀 Live Demo
https://dpsh613.github.io/Natour/

🛠️ Tech Used
HTML5
CSS3
Sass (SCSS)
Responsive Design (Media Queries)
CSS Animations
BEM Methodology

✨ Key Features
Advanced Sass architecture (7–1 pattern)
BEM naming convention
Pure CSS popup modal
CSS clip-path creative sections
Custom grid system using floats
Smooth animations and transitions
Fully responsive across devices

📚 What I Learned
Structuring large CSS projects with Sass
Writing reusable and maintainable CSS using BEM
Building complex layouts without frameworks
Creating animations using only CSS
Using clip-path and shape-outside for creative design
